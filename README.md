# 🚀 CareerMatch AI

An AI-powered career recommendation platform that analyzes resumes, extracts skills, matches users with suitable job roles, and generates personalized learning roadmaps to bridge skill gaps.

## 🌟 Features

### 📄 Resume Analysis
- Upload PDF or TXT resumes
- Extract technical skills automatically
- Detect experience level (Junior, Mid, Senior)
- Parse resume text using NLP techniques

### 🎯 Smart Job Matching
- AI-driven job recommendations
- Skill-based matching engine
- Match percentage scoring
- Experience-level filtering
- Missing skills identification

### 📚 Personalized Learning Roadmaps
- Learning paths for missing skills
- Recommended courses
- Suggested books
- Hands-on project ideas
- Estimated learning hours

### 🛠 Skill Management
- Add custom skills manually
- Remove unwanted skills
- Dynamic skill cloud visualization
- Real-time recommendation updates

### 💻 Modern UI
- Responsive dashboard
- Glassmorphism design
- Interactive job cards
- Animated skill tags
- Dark theme interface
